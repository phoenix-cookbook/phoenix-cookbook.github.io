# 간략한 소개

Phoenix (opens new window) 웹 프레임워크를 이용하여 누구나 쉽게 웹 어플리케이션을 제작할 수 있도록 하기 위해 개설한 튜토리얼 문서입니다.

특정 프레임워크에 대한 경험이 있지 않더라도, 누구나 Phoenix를 이용해서 나만의 웹 어플리케이션을 만들 수 있도록 하고 싶습니다. 뜻을 같이 하고 싶으시다면 언제든지 Issue & PR 부탁드립니다!
